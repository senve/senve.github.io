几个易用的梯子

发表于 7月 21日 2013

去年在老的 WordPress 搭的博客上发过一篇关于翻墙的文，简单链接了几个地址，介绍了几种我在用的梯子，新博客出来之后，觉得那文已经算是失效了，尤其是针对 iOS 平台的梯子，现在出来了不少好用的代理，所以也就没搬过来，决定再写一篇介绍下现在我在用的梯子，当然易用是第一点，同时稳定也是不可少的。

电脑端

爬过墙的应该都会听说过或者使用过免费的 GoAgent 这个工具，我入新电脑后的第一件事，就是利用它先部署一个自己觉得方便的网络环境，也就是搭好梯子，因为我在电脑端最离不开的就是谷歌，谷歌被墙后，真切地体会到不方便，尤其在需要查阅一些感兴趣的东西的时候，比如搭建这个基于 OctoPress 的博客，对于我这么一个小白，依靠的就是不断地谷歌。

关于这个工具，Windows 用户可参考该官方教程，注意的两点就是：证书的导入和Chrome 或者 Firefox 相应插件的设置。

倘若你是在 Windows 上用过此工具，转到 Mac OS X 后还想使用它，那么推荐使用GoAgentX，此工具还内置支持西厢第 3 季、SSH、stunnel、shadowsocks 这几种方式的代理，支持登录后自动启动，自动检查更新等。关于使用它内置 GoAgent 设置代理搭配 Firefox 翻墙的教程可以参考这篇教程，Chrome 用户可变通一下，搭配上面 Windows 下官方的教程，就是注意下证书的导入就可。

此外在 Mac OS X 下的梯子还推荐Break Wall，类似 GoAgentX 内置了多重代理方式，当然也内置支持 GoAgent 代理，使用更为方便地多，且官方说明详细，程序更新较积极，除了界面不如 GoAgentX 那么友好外，其他感觉都挺好。

移动端

这里的移动端狭隘地只表示手机与平板电脑，以往针对 Android 和 iOS 的梯子，都是通过使用 VPN 实现，但用过 VPN 的朋友都知道，这是一种全局代理的方式，所以访问国内的一些站点速度会相应较慢，访问优酷这类视频网站，还会被识别成国外的 IP 不能看视频。

后来出现了不少比 VPN 方便许多的代理方式，通过设置服务商提供的 Pac 便可实现代理，这类的服务，我用过的有曲径、iAkross以及Jayproxy。

其中曲径最为好用，购买后一次设置完，以后就完全不用去管它，且还支持后台自定义代理规则，比如 A 站曲径默认是不代理了，通过自定义后，再访问 A 站，就是通过曲径代理了，这是较为人性化的地方，而且还有比较快的速度，iOS 下能流畅播放油管的视频，缺点就是价格，经过几次服务升级后，现在的曲径普通在 iOS 上的代理服务已经达到 25 块一月，这对我这种刷 Twitter 都不太频繁的人，无疑购买是种浪费。

再者就是Jayproxy和 iAkross 了，这俩工具也是设置好 pac 后，访问如 Twitter、Facebook 这类站点会自动谈出一个登录框，输入自己在此站点注册的帐号密码后，效果与曲径类似，以后几乎也就不用去管了，只是弹框会在代理验证失效时出现，速度也比一般的 VPN 要快，价格更是比曲径便宜不少。

电脑 & 移动端

GoAgent 是支持电脑又支持越狱后 iOS 的多平台工具，但是其在 iOS 上的体验非常一般。曲径前不久开通了同时支持移动端和电脑端的服务，但一个月的价格竟然达到 50 块，显然对于一般只想搬个梯子便利谷歌或看个视频的用户来说，50 块一月是比较贵的，但通过 iOS 服务的体验，我觉得它们的速度肯定是不会差，自然就是花更多的钱买更好的服务。

拿我自己来说，我 iOS 上使用的是曲径的服务，但是这几天就要到期了，是否续费还在考虑，还有两台在用的电脑，一台 Windows 一台 OS X，均使用的是 GoAgent 的免费代理，之前用 GoAgent 看油管还是挺给力的，可这几个速度很不尽如人意，尤其这几天用 GoAgent 代理的时候，谷歌搜索竟然频繁出现“Google sorry”、“跳验证码”，自然我就想换梯子了。

看到Jayproxy的官方微博是一个偶然，第一个吸引我的就是代理方式，类似曲径，通过 pac，也无需越狱，看评价也不错，比较好的是可以免费试用一天再决定买不买。

然后，多的也就不说了，Jayproxy一个帐号便支持 iOS、Android、Windows、Mac OS X 四大平台，设置使用非常方便，具体可下载官方教程设置，最重要的是，它解决了我在 Mac OS X 下看油管视频的问题，现在 720P 的视频几乎可流畅播放，如果是 480P 的话，是完全流畅，用谷歌的问题也没有再出现，试用了一天，没犹豫就买了 3 个月，价格是真的比曲径便宜不少，速度上却几乎无差别，缺点就是不支持自定义代理列表，但可在微博联系其官博加入。

Jayproxy，你可以先注册试用一天，使用过后再决定买不买，可直接点我的推荐链接，或者在官网注册时，方便地话，推荐人填上：senve316，我会在你购买其服务时，相应得到一定的天数奖励，当然不填不要紧，发这篇文的初衷还是充实自己新搭的博客，想想关于梯子的话题在国内还是较热门的，发篇这种博文也算合适。

最后也简单谈下翻墙吧

个人认为，墙的存在，影响最大的就是我们这类还可以学习汲取外界营养的年轻人，有太多的好东西竟然被阻隔墙外，这我想是设墙者未曾想过的事，墙外的世界谈不上美好，只能说更广阔，今天看了几集只在 YouTube 上的免费学习视频教程，愈发觉得那是一个宝库，翻墙的理由不应该只是在推特、脸书上扯淡，花钱或者是花功夫搬梯子过去了，那么就应该考虑下收益，翻墙不是门技艺，拿来装逼更显可笑，可能是一种无拘无束吧。

在一堵墙前，如果觉得墙内的时间不够精彩，就秉着好奇心，搬个梯子，站起来看看吧。
