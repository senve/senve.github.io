可劲折腾的Octopress

发表于 7月 14日 2013

前言

玩 Octopress 和当初玩 WordPress 开始的时候一样，扎进了选主题和改主题里，一天的时间在这找与改中流逝之快令己感叹不已，遂看博客壳子还算成形，也想到建博客的根本目的就是有个好的个人发布文字的环境，那么现在看看也是达成了，所以，新的开始，就先谈谈我的新博客吧。

正题

我的博客所用到的东西：

  1. 博客框架：Octopress（用 MarkDown 语法写作，依靠指令完成静态页部署等）
  2. 托管：Stdyun（Github在国内的访问速度很一般，而这个托管服务不但快很多还是免费的）
  3. 博客主题：Object（自己改了字体并精简了一些东西，更清爽了）
  4. 评论系统：Disqus（Octopress 内置这个，比多说看起来更舒适些）
  5. 图床：围脖是个好图床（特别好使，谁用谁知道）
  6. 写作工具：Mou（这么好用的东西竟然是免费的）

以上这么几样，外加 Alfred 的几个 Workflow 和 TextExpander 提高效率外，再依靠 OS X 内自带的终端管理器，上手后很舒适的写作感觉。

然后博客的第一篇博文是引用外链的 OS X 下搭建 Cctopress 的教程，细心的来访者会发现，点击博文标题后，却是跳转到了另外一个博客，这里用到的是 Linklog，用这种方式发博文可以拿来推荐、补充等，方法教程可以参考这篇博文：Make a Linked List With Octopress。

收尾

很多人可能与我类似，读的不是计算机专业身份也不是码农，一个小白看不懂代码却对它有不少兴趣，Octopress 恰恰满足了我这样的需求，全程通过代码指令搭建，看着一行行命令的反馈，这个过程非常过瘾，搭建成功后，连写博文、更新管理博客等等还是用代码，这样的博客创作体验，伪 Geek 表示很暗爽。
