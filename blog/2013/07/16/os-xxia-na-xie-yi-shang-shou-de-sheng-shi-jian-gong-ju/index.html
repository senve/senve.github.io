OS X下那些易上手的省时间工具

发表于 7月 16日 2013

接触玩 OS X 的时间可以说还比较短，但在这么短短的一段时间里，就已爱上这种体验，而理由就是在 OS X 下，有让我容易上手的效率类小工具，使用它们，提高效率的同时自然能省下不少时间。

下面简单介绍几个我在用的，针对新手 Mac 用户又容易上手的效率类小工具。

  • Alfred：免费版，可直接当浏览器地址栏来用，默认快捷键option+ 空格呼出后直接输入网址回车就可直达站点，输入搜索关键字可跳转至谷歌搜索，并还支持自定义搜索，其他与 OS X 自带的 Spotlight 类似，但其真正强大的地方在于花 15 英镑购买 Powerpack 后，便可安装扩展各类的 Workflow 插件，v2 版更新已经有一段时间，各类的 Workflow 插件十分丰富，开发者还可自己开发插件，普通用户可在官方论坛下载自己需要的插件，购买 Powerpack 后的 Alfred，可以称得上是一柄省时利器。
  • PopClip：选择一段文字后，会弹出类似 iOS 下的复制、粘贴等选项条，相对 Alfred 的 Powerpack 价格，此工具可谓白菜价，区区 5 美刀但也是支持插件扩展，除了官方插件外，还支持其他开发者写的插件。工具之间相互搭配也是提高操作效率的办法，比如使用 Alfred 的Make PopClip Appear插件使得 PopClip 更为简单易用。
  • jitouch：OS X 下的触控板因为支持手势与多点触控本就十分好用，可安装此工具后，手势不但可自定义，支持的手势功能也更为丰富，此工具还能扩展 Apple 的 Magic Mouse 手势，可免费试用，购买注册也不贵，推荐。有了它，不打游戏的话，特别是浏览网页，不自觉就会把手从鼠标移动到触控板上。
  • Cinch：可拖动窗口往左、右或上快速调节定位窗口大小，官网有演示视频，免费版除了一段时间会弹一次窗之外，可以说已经是完全不影响使用，但花 7 刀去弹窗又能支持开发者也可谓划算。
  • CheatSheet：OS X 下的快捷键一般都离不开command 键，功能和 Windows 下的contrl 键触发快捷操作类似，但记快捷键是吃力不讨好的事，尤其对于新手 Mac 用户来说，安装此工具后，按住command 键即可弹出此应用下支持的快捷键指令，官方免费。
  • ClipMenu：记录剪贴板的复制历史，当需要 copy 不少东西时，可尽管都先复制下来，然后回到需要粘贴的地方，通过它一个个快速粘贴下来，除了复制文字外，还支持富文本、PDF、图片等，免费的实用小工具。
  • Droplr：分享类工具，安装后，菜单栏会显示其应用图标，可快速通过拖动要分享的内容到该图标上建立分享链接，支持文件、图片、视频、链接等分享，免费用户赠送 1G 空间容量，付费用户可扩容到 100G 还赠送个性域名定制。
  • Dropbox：非常著名的云端储存服务商，iCloud 还未出现前，一些软件的同步都是靠它实现，OS X、Windows、iOS 等均有相应的客户端，同步储存文件稳定可靠，也可如 Droplr 一样建立链接分享文件，我习惯用它储存一些重要的小型文档，同步到各平台，只要一个帐号就可以快速浏览。
  • Dash：该应用的定位群体是程序员开发者，因为提供了各式各类的 API 文档下载，具体支持的文档可看官网介绍，本地快速搜索。而我作为普通用户，感叹它的 Snippet 功能真的是太方便，与TextExpander相比，功能虽然少了一点，但同样支持占位标识，使用方法也与 TextExpander 类似，内建好自定义语法片段后，可快速提高输入效率。Mac App Store 内可免费下载试用全部功能，促销阶段只需花 8 刀完成内购后不再弹框，相对 TextExpander 的 34.95 刀，对于普通用户来说，可谓合算至极。

由于 Alfred 和 PopClip 可插件扩展的特性，以及 Dash 的片段编写，必然决定了它们使用方向的多元化，尤其针对 Alfred 而言，如此利用丰富的 Workflow 插件为自己提升效率，完全取决于个人。本来想写的第一篇博文就是关于我使用的 Aflred 插件的，可这类文一谷歌一大堆，也没有多大意义，更何况对于 OS X 的软件使用，我更还只是个入门的初学者而已。

这些工具不是我说好用它就是好用的，人与人之间差异性的存在，导致对于同一样东西，感知体会可能是不同的，故以上软件只是作为推荐参考，但是这些软件都是可试用后再付费或者就是免费的，所以推荐都试试。
