初衷

几个易用的梯子

发表于 7月 21日 2013

去年在老的 WordPress 搭的博客上发过一篇关于翻墙的文，简单链接了几个地址，介绍了几种我在用的梯子，新博客出来之后，觉得那文已经算是失效了，尤其是针对 iOS 平台的梯子，现在出来了不少好用的代理，所以也就没搬过来，决定再写一篇介绍下现在我在用的梯子，当然易用是第一点，同时稳定也是不可少的。

电脑端

爬过墙的应该都会听说过或者使用过免费的 GoAgent 这个工具，我入新电脑后的第一件事，就是利用它先部署一个自己觉得方便的网络环境，也就是搭好梯子，因为我在电脑端最离不开的就是谷歌，谷歌被墙后，真切地体会到不方便，尤其在需要查阅一些感兴趣的东西的时候，比如搭建这个基于 OctoPress 的博客，对于我这么一个小白，依靠的就是不断地谷歌。

关于这个工具，Windows 用户可参考该官方教程，注意的两点就是：证书的导入和Chrome 或者 Firefox 相应插件的设置。

倘若你是在 Windows 上用过此工具，转到 Mac OS X 后还想使用它，那么推荐使用GoAgentX，此工具还内置支持西厢第 3 季、SSH、stunnel、shadowsocks 这几种方式的代理，支持登录后自动启动，自动检查更新等。关于使用它内置 GoAgent 设置代理搭配 Firefox 翻墙的教程可以参考这篇教程，Chrome 用户可变通一下，搭配上面 Windows 下官方的教程，就是注意下证书的导入就可。

此外在 Mac OS X 下的梯子还推荐Break Wall，类似 GoAgentX 内置了多重代理方式，当然也内置支持 GoAgent 代理，使用更为方便地多，且官方说明详细，程序更新较积极，除了界面不如 GoAgentX 那么友好外，其他感觉都挺好。

可劲折腾的Octopress

发表于 7月 14日 2013

前言

玩 Octopress 和当初玩 WordPress 开始的时候一样，扎进了选主题和改主题里，一天的时间在这找与改中流逝之快令己感叹不已，遂看博客壳子还算成形，也想到建博客的根本目的就是有个好的个人发布文字的环境，那么现在看看也是达成了，所以，新的开始，就先谈谈我的新博客吧。

正题

我的博客所用到的东西：

  1. 博客框架：Octopress（用 MarkDown 语法写作，依靠指令完成静态页部署等）
  2. 托管：Stdyun（Github在国内的访问速度很一般，而这个托管服务不但快很多还是免费的）
  3. 博客主题：Object（自己改了字体并精简了一些东西，更清爽了）
  4. 评论系统：Disqus（Octopress 内置这个，比多说看起来更舒适些）
  5. 图床：围脖是个好图床（特别好使，谁用谁知道）
  6. 写作工具：Mou（这么好用的东西竟然是免费的）

以上这么几样，外加 Alfred 的几个 Workflow 和 TextExpander 提高效率外，再依靠 OS X 内自带的终端管理器，上手后很舒适的写作感觉。

然后博客的第一篇博文是引用外链的 OS X 下搭建 Cctopress 的教程，细心的来访者会发现，点击博文标题后，却是跳转到了另外一个博客，这里用到的是 Linklog，用这种方式发博文可以拿来推荐、补充等，方法教程可以参考这篇博文：Make a Linked List With Octopress。

收尾

很多人可能与我类似，读的不是计算机专业身份也不是码农，一个小白看不懂代码却对它有不少兴趣，Octopress 恰恰满足了我这样的需求，全程通过代码指令搭建，看着一行行命令的反馈，这个过程非常过瘾，搭建成功后，连写博文、更新管理博客等等还是用代码，这样的博客创作体验，伪 Geek 表示很暗爽。
